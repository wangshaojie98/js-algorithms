## 栈
括号问题在面试中出现频率非常高， 这类题目我们一般首选用栈来做。

为什么可以用栈做？大家想想，括号成立意味着什么？意味着对称性。

巧了，根据栈的后进先出原则，一组数据的入栈和出栈顺序刚好是对称的。比如说1、2、3、4、5、6按顺序入栈，其对应的出栈序列就是 6、5、4、3、2、1：
```
123456
654321

"()"
")("
```

对称关系一目了然。

因此这里大家可以记下一个规律：题目中**若涉及括号问题，则很有可能和栈相关**。
